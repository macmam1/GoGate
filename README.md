## 📌 Project Goals

GoGate is designed to provide:
- Stable connectivity workflows in high-filtering / unstable network conditions
- Smart profile selection and fallback strategies
- Clean, modern UI with performance-aware themes
- Cross-platform architecture with shared core logic

Initial target platforms:
- **Windows** (phase 1)
- **Android** (phase 1)
- macOS/Linux (phase 2+)

---

## ✨ Core Capabilities (Planned)

- Subscription and configuration management
- Health probes (quick/deep) + quality scoring
- Geo/ISP-aware server grouping
- Smart connect + multi-step fallback chain
- Structured logs + diagnostics export
- Performance themes: **Lite / Balanced / Rich**
- Accessibility mode: **Reduced Motion**

### Advanced/Restricted Network Modes
Advanced modes are designed with:
- explicit user opt-in,
- clear risk notice,
- immediate disable path (kill-switch),
- local audit trail.

> These modes are meant for lawful, user-controlled connectivity compatibility.
