# GoGate — Program Blueprint (No-Code Phase)

> وضعیت فعلی: **فاز طراحی و برنامه‌ریزی** — هنوز کدنویسی شروع نشده است.
>
> Branding Signature: **MRH-DevLoop**

این پوشه برای مدیریت حرفه‌ای پروژه ساخته شده تا مسیر ساخت برنامه به‌صورت شرکتی، قابل رهگیری و بدون بی‌راهه انجام شود.

## هدف محصول
ساخت یک کلاینت چندسکویی (Windows/Android در فاز اول؛ macOS/Linux در فاز بعدی) برای مدیریت و اتصال به کانفیگ‌های پروکسی/VPN با:
- مدیریت ساب و کانفیگ
- ارزیابی کیفیت اتصال
- fallback هوشمند
- UI حرفه‌ای و انیمیشنی
- انتشار منظم در GitHub Releases

## نکته انطباق و امنیت
این پروژه فقط در چهارچوب استفاده قانونی، مجاز و با رضایت کاربر طراحی می‌شود.
- قابلیت‌های ریسکی فقط با **opt-in**، هشدار صریح، و ثبت پذیرش مسئولیت کاربر فعال می‌شوند.
- از ارائه/پیاده‌سازی منطق تهاجمی یا سوءاستفاده‌پذیر جلوگیری می‌شود.

## ساختار
- `docs/` اسناد محصول، معماری، ریسک، نقشه راه
- `management/` تسک‌ها، تصمیمات، صورت‌جلسه‌ها

## خروجی این فاز
1. PRD + Feature Dependency Matrix
2. Architecture + Security Guardrails
3. Roadmap فازبندی شده
4. GitHub Repo/Release Playbook
5. User action checklist برای اتوماسیون حداکثری
