# GoGate

> **کلاینت تطبیق‌پذیر چندسکویی برای اتصال در شبکه‌های محدود و ناپایدار**  
> تمرکز اصلی: پایداری، کنترل کاربر، و معماری حرفه‌ای.  
> امضای برند: `MRH-DevLoop`

---

## 🎯 هدف پروژه

GoGate برای این ساخته می‌شود که:
- در شرایط فیلترینگ/اختلال شدید، اتصال پایدارتر ارائه دهد
- مدیریت کانفیگ و ساب را حرفه‌ای و قابل رهگیری کند
- با fallback هوشمند، قطع و وصل‌ها را کاهش دهد
- رابط کاربری مدرن، روان و بهینه داشته باشد

پلتفرم‌ها:
- **ویندوز** (فاز ۱)
- **اندروید** (فاز ۱)
- macOS/Linux (فازهای بعدی)

---

## ✨ قابلیت‌های کلیدی (برنامه‌ریزی‌شده)

- مدیریت لینک ساب و کانفیگ
- تست سلامت سریع/عمیق + امتیازدهی کیفیت
- دسته‌بندی بر اساس ISP/موقعیت
- اتصال هوشمند + زنجیره fallback
- لاگ ساختاریافته + خروجی عیب‌یابی
- تم‌های عملکردی: **Lite / Balanced / Rich**
- حالت **Reduced Motion** برای سیستم‌های ضعیف‌تر

### حالت‌های پیشرفته/شبکه محدود
این حالت‌ها فقط با این شرایط فعال می‌شوند:
- تایید صریح کاربر (opt-in)
- نمایش هشدار و توضیح ریسک
- امکان خاموش‌سازی فوری
- ثبت محلی پذیرش مسئولیت
