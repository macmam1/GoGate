# 05) User Responsibilities & Access Checklist

## دسترسی‌هایی که برای شروع از شما لازم است
- [ ] نام نهایی پروژه
- [ ] نام کاربری GitHub
- [ ] نوع ریپو (Public/Private)
- [ ] PAT محدود به همان ریپو
- [ ] تصمیم درباره پلتفرم‌های فاز اول (Windows + Android تایید شد)
- [ ] فهرست اولیه پروتکل‌ها/engineها

## Secrets Policy
- token فقط در secrets manager
- عدم ارسال رمز/Recovery code در چت
- زمان انقضا برای token
- پس از تحویل نسخه اولیه: token rotate/revoke

## تصمیمات محصولی که باید نهایی شوند
1. UI language: فارسی/انگلیسی/دو زبانه
2. اولویت پروتکل‌های فاز 1
3. مسیر آپدیت (in-app updater یا فقط GitHub Releases)
4. سیاست telemetry (خاموش پیش‌فرض یا opt-in)
